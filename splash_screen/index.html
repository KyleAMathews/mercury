Setup Instructions

Ready to run Drupal liquid metal fast?

Congradulations on starting a Mercury instance. This release contains the latest Pressflow Drupal (core version 6.14) as well as the rc1 release of CacheRouter module.

Some quick links to get you oriented to the system:

  • Your Pressflow install (user/login = admin/drupal, change this now!)
  • Phpinfo() (get acquainted with the php config)
  • Apc.php (check out your current cache stats)

In order to get things up and running for production, you should attach and mount an EBS and move your mysql database there. Also, you can delete this index.html file and move the contents of the pressflow directory directly to /var/www (or update /etc/apache2/sites-available/default to use /var/www/pressflow as the docroot).

For more information, stay tuned to getpantheon.com and our related Drupal and Launchpad groups!
